## 数据特点

### 血糖数据
- 包含7种测量类型：空腹、早餐后、午餐前、午餐后、晚餐前、晚餐后、睡前
- 每天随机生成1-3条记录
- 约10%的概率生成异常值（血糖过高或过低）

### 血压数据
- 每天随机生成1-2条记录
- 约50%的概率生成异常值（血压过高或过低）
- 包含心率数据（60-100次/分钟）

## 如何查看

### 1. 在小程序中查看
- **首页**: 可以看到今日的血糖和血压记录、异常预警、最近记录
- **血糖趋势**: 显示30天内血糖变化趋势图，可按测量类型筛选
- **血压趋势**: 显示30天内血压变化趋势图（收缩压/舒张压双线图）
- **历史记录**: 查看所有历史记录，支持按时间筛选（全部/今天/近7天/近30天）

### 2. 在控制台查看
运行测试脚本查看详细数据：
```bash
node miniprogram/testData.js
```

### 3. 数据文件
- `utils/mockData.js`: 测试数据生成器，可导出生成的数据
- `testData.js`: 独立的测试数据查看脚本

## 异常判断标准

### 血糖异常
- 空腹血糖 > 7.0 mmol/L 或 < 3.9 mmol/L
- 餐后2小时血糖 > 11.1 mmol/L 或 < 3.9 mmol/L

### 血压异常
- 收缩压 > 140 mmHg 或 < 90 mmHg
- 舒张压 > 90 mmHg 或 < 60 mmHg
